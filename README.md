# PhD Life - A Stress Relief Game

一个关于博士生活的 2D 平台游戏，通过游戏化方式展现读博过程中的压力与欢乐。

## 游戏概念

- 通过游戏化方式重现读书过程中的压力和挑战
- 将学术压力转化为可互动的游戏元素
- 通过游戏体验来达到解压效果

## 游戏特点

- 2D 平台跳跃游戏
- 像素风格画面
- 收集和躲避机制
- 压力值系统

## 开发环境

- Godot 4.4
- 开发语言：GDScript

## 如何运行

1. 安装 Godot 4.4 或更高版本
2. 克隆本仓库
3. 使用 Godot 打开项目文件夹
4. 点击运行按钮或按 F5 启动游戏

## 游戏控制

- 左右方向键：移动
- 空格键：跳跃
